CIT 281 Project 1 - CLI Application

This project demonstrates basic command line applications in JavaScript.

Files:

  • p1-date.js - Displays current date and time
  • p1-random.js - Generates random numbers

How to run:

node p1-date.js
# Outputs current date and time

node p1-random.js
# Generates a random number
